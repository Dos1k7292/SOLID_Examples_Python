# SOLID Examples in Python

Примеры реализации принципов **SOLID** на Python:

1. **SRP (Single Responsibility Principle)** — `srp/order.py`
2. **OCP (Open-Closed Principle)** — `ocp/employee_salary.py`
3. **ISP (Interface Segregation Principle)** — `isp/printers.py`
4. **DIP (Dependency-Inversion Principle)** — `dip/notifications.py`

Каждый пример можно запускать отдельно.
